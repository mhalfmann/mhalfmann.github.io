Materialerstellung auf Code-Basis

Material kann mit einem Code-Editor wie z.B. Visual Studio Code erzeugt werden.

  • Material besteht aus HTML-Dateien, Beispiel unten
  • Pflichtelemente (grün) sind in jeder HTML-Datei zur Intergration der Versuchsumgebung notwendig und immer identisch
  • Seitenüberschrift (violett) einfach ersetzbar
  • Inhalt (rot) kann aus Vorlagen kopiert und angepasst werden
